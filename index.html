Projects

Disaster Recovery Plan for KCTCS

In this project, I developed a disaster recovery plan for the Kentucky Community and Technical College System (KCTCS). The plan outlines the steps to be taken in the event of a disaster, including the identification of critical systems, the creation of a backup and recovery strategy, and the communication and coordination with stakeholders.

DOS Tools

In this project, I tested a suite of tools for detecting distributed denial of service (DDOS) attacks. The tools include a network sniffer, a packet filter, and a traffic generator, which can be used to monitor network traffic, identify malicious patterns, and simulate attacks.

Malware Report

In this project, I researched and analyzed the different types of malware and their impact on security. The report covers the history of malware, its evolution, and the motivations behind its use. This project showcases my understanding of malware and its role in modern security.

Network Encryption

In this project, I studied the various cryptography standards and protocols used to protect data as it travels across networks. The project covers encryption at the application, network, and link layers, and includes an overview of common standards such as SSL, IPsec, and WPA2. This project demonstrates my knowledge of network encryption and its importance in securing data.

Social Engineering

In this project, I researched and analyzed the different tactics and techniques used in social engineering attacks. The report covers common types of social engineering, such as phishing and pretexting, and provides tips and best practices for defending against these attacks. This project demonstrates my understanding of social engineering and its impact on security.

Web Application Security Threats

In this project, I researched and analyzed the different threats to web application security. The report covers common types of attacks, such as SQL injection and cross-site scripting, and provides recommendations for securing web applications. This project showcases my understanding of web application security and its importance in protecting data and systems.

C# Console Apps

This project involves the development of programs using the C# programming language. The programs are designed to demonstrate a range of fundamental programming concepts, including data types, control structures, simple data structures, error handling, object-oriented programming, graphical user interfaces, and modular programming.

Database Project

This project provides an overview of database and database management system concepts, including internal design models, normalization, network data models, development tools, and applications.
